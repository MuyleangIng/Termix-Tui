# Fonts

Termix recommends Nerd Fonts for Oh My Posh icons, but missing fonts do not block startup.

Recommended fonts:

- CaskaydiaCove Nerd Font
- CascadiaCode Nerd Font
- JetBrainsMono Nerd Font
- FiraCode Nerd Font
- MesloLGM Nerd Font
- Hack Nerd Font
- UbuntuMono Nerd Font

Useful commands:

```bash
termix fonts list
termix fonts install "JetBrainsMono Nerd Font" --yes
termix fonts apply "JetBrainsMono Nerd Font" --windows-terminal
```

Fallback stack:

```text
CaskaydiaCove Nerd Font
Cascadia Code
JetBrains Mono
Fira Code
Consolas
Courier New
monospace
```
