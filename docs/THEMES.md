# Themes

Termix imports official Oh My Posh themes from:

https://github.com/JanDeDobbeleer/oh-my-posh/tree/main/themes

Useful commands:

```bash
termix themes update
termix themes
termix cache rebuild
termix themes apply catppuccin_mocha --profile "PowerShell 7"
```

Cache metadata can be cleared safely:

```bash
termix cache clear
```

Clearing cache does not delete downloaded theme files.
