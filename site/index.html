[ TERMINAL WORKSPACE ]

Manage your terminal like a TUI dashboard.

Termix applies Oh My Posh themes, shell profile blocks, Nerd Fonts, and Windows Terminal settings from one focused CLI/TUI workflow.

Get Started View GitHub

[ FEATURES ]

Built for shell setup, not manual profile editing.

[ THEME MANAGER ]

Download official Oh My Posh themes, browse them, and apply one to the selected profile.

[ PROFILE MANAGER ]

Write safe managed blocks for PowerShell 7, Windows PowerShell, Git Bash, WSL, Fish, and Nushell.

[ FONT SETUP ]

Install and apply a Nerd Font with fallback handling when glyphs are missing.

[ CROSS-SHELL ]

Keep shell-specific profile paths separate so one profile fix does not break another shell.

[ LIVE PREVIEW ]

Use real Oh My Posh theme data in the terminal workflow before applying a prompt.

[ BACKUP & RESTORE ]

Back up profile files before replacing Termix-managed blocks and keep user content intact.

[ WORKSPACE ]

A homepage preview that feels like Termix.

The preview uses the same dark panels, cyan borders, status chips, and activity-log rhythm as the TUI without copying a screenshot.

[ INSTALL ]

Install, open a new terminal, run setup.

The Windows installer downloads the latest release and bootstraps Oh My Posh, the default Nerd Font, and official themes.
